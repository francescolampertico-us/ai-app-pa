# Style Reference Notes - Submitted Memo

Use the submitted memo as the canonical reference for the default output family.

## Key takeaways

- long-form SUBJECT line
- centered standalone hearing title
- high-level Hearing Overview
- exact heading `Committee Leadership Opening Statements`
- exact heading `Witnesses Introductions and Testimonies`
- Q&A grouped by member headings
- short, memo-like paragraphs
- no standalone closing section by default
- single confidentiality line in content; repeated footers handled in export

## Important caution

The submitted memo also reinforces the need for strict metadata verification.
Even strong finished memos can contain hearing-date, memo-date, or day-of-week inconsistencies.
The verifier should catch these instead of assuming the reference memo is fact-perfect.
