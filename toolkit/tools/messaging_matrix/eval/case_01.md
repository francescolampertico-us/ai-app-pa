# Eval Case 01: AI Safety Bill with Full Context

## Input
- Position: "Support the AI Safety and Innovation Act — mandatory pre-deployment safety testing protects consumers without stifling innovation."
- Context: Bill summary with specific provisions (compute threshold, NIST institute, $500M authorization, liability safe harbor)
- Organization: "TechForward Alliance"
- Variants: all

## Pass Criteria

### Message House
- [ ] Core message is one sentence and quotable
- [ ] Exactly 3 pillars with distinct argument dimensions
- [ ] Proof points reference specific facts from the context (e.g., "$500M", "10^26 FLOP", "19-7 vote")
- [ ] No fabricated statistics or provisions
- [ ] Key terms are specific and consistent

### Hill Talking Points
- [ ] 300-500 words
- [ ] Bulleted format
- [ ] Contains a clear "ask" (specific action requested)
- [ ] Includes anticipated pushback section
- [ ] References specific bill provisions

### Press Statement
- [ ] 300-500 words
- [ ] Includes FOR IMMEDIATE RELEASE header
- [ ] Contains a liftable quote attributed to the organization
- [ ] Standard press release structure

### Social Media
- [ ] Three separate posts (X, LinkedIn, Facebook)
- [ ] X post under 280 characters
- [ ] LinkedIn post 150-300 words with hashtags
- [ ] Each post stands alone

### Grassroots Email
- [ ] 150-300 words
- [ ] Has a subject line
- [ ] Contains a specific call to action
- [ ] Uses "you" language

### Op-Ed
- [ ] 500-800 words
- [ ] First-person voice
- [ ] Opens with a news peg, not a generic statement
- [ ] Addresses counterarguments
